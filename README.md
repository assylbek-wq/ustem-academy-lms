# USTEM Academy LMS

Полнофункциональная Learning Management System для **USTEM Academy** — образовательной академии STEM в Казахстане.

Платформа полностью **статическая** — HTML + Tailwind CSS + Vanilla JS + LocalStorage. Никаких сборщиков, бэкенда или баз данных.

> 💡 Если ты пришёл сюда продолжать работу в **Claude Code** — обязательно прочти [CLAUDE.md](CLAUDE.md). Там вся архитектура, договорённости и история решений.

## Быстрый старт

```bash
# 1. Перейти в папку проекта
cd LMS

# 2. Запустить локальный сервер (Python обычно уже стоит)
python3 -m http.server 8000

# 3. Открыть в браузере
open http://localhost:8000
```

> ⚠️ **Важно:** не открывайте `index.html` напрямую через `file://` — YouTube/Vimeo не работают в embedded iframe без HTTP-сервера. Получите «Ошибка 153» от плеера.

## Основные возможности

### Для студента
- Каталог курсов с поиском и фильтрами (категория, уровень, сортировка)
- Запись на курс одним кликом
- Плеер уроков с навигацией и боковой панелью прогресса
- Квизы: вопросы с 4 вариантами, мгновенная проверка, нужно 60% правильных
- Дашборд: активные/завершённые курсы, статистика, **HTML-сертификаты**
- Профиль с историей квизов

### Для админа
- **CRUD курсов** через визуальный редактор `admin-edit.html`:
  - Quill WYSIWYG для содержимого уроков (форматирование, цвета, выравнивание)
  - **Шаблоны блоков** одним кликом: 📘 введение, 💡 совет, ⚠️ предупреждение, 📊 таблица, 💻 код, 🎬 видео, 🖼 картинка, 📎 файл
  - **Видео embed** из 8 платформ: YouTube, Vimeo, Rutube, **Kinescope**, **VK Video**, **OK.ru**, **Dzen**, прямые MP4
  - **Drag-and-drop** картинок прямо в редактор
  - **Подсветка кода** через Prism.js — 13 языков (Python, JS, C++, Java, Bash, SQL и др.)
  - **Авто-распознавание URL** при paste — встройте видео без захода в меню
- **Drag-and-drop** для порядка уроков
- **Редактор квизов** с радио для правильного ответа
- **Языковые вкладки** РУ/ҚАЗ/EN с индикаторами заполнения (🟢🟡🔴)
- **Статус** Черновик / Опубликован
- **Авто-сохранение** каждые 30 сек + Ctrl+S
- **Экспорт/Импорт JSON** для бэкапа и переноса
- **Дублирование** курса/поста одним кликом
- **Поиск** по курсам и постам в админке

### Дополнительно
- **Сертификаты** — HTML-документ с подписью студента, скачивается с дашборда
- **Multi-tenant ready** — структура подходит для нескольких академий
- **Адаптив** под мобильные

## Деплой

Сейчас задеплоено на Netlify: `silly-biscochitos-290aa3.netlify.app`.

Поскольку проект статический — деплоится куда угодно:

**Netlify** (drag-and-drop)
1. Зайди на [netlify.com](https://app.netlify.com/drop)
2. Перетащи папку `LMS/` в окно браузера
3. Получи URL вида `https://random-name.netlify.app`

**Через Git** (Vercel, Netlify, Cloudflare Pages)
- Подключи репозиторий
- Build command: пусто
- Publish directory: `/` или путь к папке `LMS/`

**Любой хостинг**
```bash
rsync -avz LMS/ user@server:/var/www/html/
```

## Сброс данных

В админке внизу справа есть кнопка **«Сбросить базу»** — вернёт демо-курсы и удалит созданных пользователей.

Или через консоль браузера:
```js
localStorage.clear();
location.reload();
```

## Технологии

- **HTML5** + **Tailwind CSS** (CDN)
- **Vanilla JavaScript** ES6+ (без сборки)
- **Quill 2.0.2** — WYSIWYG-редактор
- **Prism.js 1.29** — подсветка синтаксиса
- **LocalStorage** — хранилище данных, версионирование схемы (`ustem_lms_vN`)
